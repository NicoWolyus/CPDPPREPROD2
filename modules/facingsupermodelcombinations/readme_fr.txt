# MODULE SUPERMODEL - améliorer votre facing sur la liste produit
#
# Disponible en version 1.6 et version 1.7
# Aucun prérequis technique nécessaire
#
# Installation, 
# Veuillez vous assurer de disposer d'une version 1.6 au moins (Vérifiez dans votre back office - en haut de l'écran, à côté du logo prestashop logo ou dans Menu / Paramètres avancés / Informations)
# 
# Télécharger votre zip via le Menu / Modules => bouton "Ajouter un nouveau module" (suivez les recommandations Prestashop)
# Une fois le zip uploadé sur votre serveur, cliqué sur Installer, puis Configurer.

# Configuration :
# Les 3 éléments doivent être configurés obligatoirement pour faire fonctionner le module en Front office : choix de la caractéristique, choix de cacher ou découvrir la caractéristique, choix de la déclinaison déclenchant le changement de produit.
# Aucun impact en back office.
#
# Lors de la configuration,
# Vous devez identifier la caractéristique assurant le point de pivot entre les produits. Vous pouvez créer des points de pivot "virtuels" en créant de nouvelles caractéristiques. Celles-ci seront récupérées automatiquement par le module.
# Vos produits devront avoir été associés à ces caractéristiques au préalable.
# La caractéristique assurant le lien entre les produits peut être masqué en front office - n'apparaïtra pas dans la partie "Détail Produit" de la fiche produit (cliquer sur "Oui" dans la partie masquer).
# Enfin, vous devez identifier l'attribut qui déclenchera le passage de produit à un autre. Choisissez l'attribut déclinaison dans la liste déroulante.
#
# Veuillez noter que le module s'applique à l'entièreté du catalogue et ne peut pas être spécifié par catégorie par exemple (ou tout autre limitation).
